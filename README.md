# R.C. Structural Design Load Preparation

A high-performance interactive structural engineering design aid for configuring multi-story building geometries, applying load overrides, and calculating factored combinations according to standard design codes.

## Local Getting Started

Ensure you have [Node.js](https://nodejs.org/) installed, then run:

```bash
# 1. Install dependencies
npm install

# 2. Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Tech Stack
* **Vite + React 19**
* **Tailwind CSS v4** (Utility-first styling grid)
* **Lucide React** (Engineering vector glyphs)
* **JSZip** (On-demand offline workspace packing)
